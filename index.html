Issue Labeler Model

The issue labeler model is the prediction engine utilized by the plugin. The model is a result of fine-tuning Google's BERT model, which is a deep learning model that excels at natural language processing, and training it for issue classification using the expansive SmartSHARK dataset. The model predicts an issue's type based on contextual analysis of its title and description.

Jira Plugin

\

The Jira plugin is an extension of the issue labeler model. It is installable into Jira Cloud instances and is viewable in an issue's page. The plugin will passively analyze an issue's title and description by communicating with the issue labeler model and display prediction types ranked by their confidence score.
